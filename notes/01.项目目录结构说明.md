# 01.项目目录结构说明

```
cms/                  集群控制端的子项目代码
msvc/                 VC++项目文件
sparkle.sln           VS解决方案文件
test/                 单元测试
DEPENDENCIES          外部依赖说明
notes/                开发文档
ref/                  标准引用
src/                  Sparkle源码
win64/                Windows-64位相关依赖
```

arch/                 平台相关支持（x86/ARM/Linux/Windows等）
common/               通用组件
consprot/             一致性协议实现
eventd/               事件驱动、网络与定时器
kvstore/              KV存储实现
msgon/                消息协议封装实现
spk-rms/              超级节点相关代码
auth/                 系统安全验证相关
compto/               压缩与加密
daemon/               守护进程
include/              模块访问入口头文件
log/                  运行日志
spk-css/              数据节点相关代码
config/               系统配置
erasurecode/          纠删码
journal/              事务日志
spk-das/              访问接入节点相关代码
main.cpp              程序入口
CMakeLists.txt        CMake文件
